# Module 3: AI-Robot Brain (NVIDIA Isaac)

## Overview

NVIDIA Isaac provides the AI capabilities that enable robots to perceive, navigate, and manipulate objects intelligently. This module covers perception pipelines, navigation stacks, and AI integration.

## Prerequisites

- Module 1 & 2 completion
- Basic machine learning concepts
- NVIDIA Jetson or CUDA-enabled GPU

## Module Structure

This module consists of the following chapters:

1. **Introduction to NVIDIA Isaac** - Platform overview and capabilities
2. **Isaac Sim** - Simulation environment for AI development
3. **Perception** - Vision and sensor processing
4. **Navigation** - Autonomous movement and path planning

## Estimated Time

**14 hours** of hands-on learning and practice

## Hardware Requirements

- NVIDIA Jetson Orin Nano (8GB minimum)
- Or desktop GPU with CUDA support (RTX 3060+)
- USB-C power supply for Jetson
- MicroSD card (64GB, UHS-I U3)
